FABFLIX


Description:

The application allows the user to login the system. User can then search for movies in the database with autocomplete search suggestions (using ajax requests). Users also have the option to browse the movie db by genre or first letter. The movie list displayed allows for pagination. Users can then add movies to their cart and input their payment info to make purchases.

This application was developed as a team (1 partner) for a course project and taught full stack development. The image to the left is a work in progress of our project.

Utilized:

Front-end: Javascript HTML CSS

Back-end: Java MySQL
